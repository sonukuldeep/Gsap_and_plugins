## Installation

The project requires node js and git.

### Clone the website using
```npm
git clone 
```

### Install this project with npm

```bash
  cd project-directory
  npm i
```

### Run using

```bash
  npm run dev
```

### In the browser open

```bash
  http://localhost:5173/
```
